# Reports Directory

This directory contains all comprehensive analysis reports generated by CodePulse.

## Report Format

Reports are saved with timestamp:
```
comprehensive_report_20250115_143022.json
                     ^^^^^^^^_^^^^^^
                     YYYYMMDD_HHMMSS
```

## Report Structure

Each report contains:
- **Project Information**: Path, date, files, lines
- **Scores**: Overall, code quality, security
- **Issues**: Categorized by severity
- **Vulnerabilities**: Security issues found
- **Recommendations**: Actionable suggestions

## Managing Reports

### List All Reports
```bash
python3 reports_manager.py list
```

### Show Latest Report
```bash
python3 reports_manager.py latest
```

### Compare Two Reports
```bash
python3 reports_manager.py compare reports/report1.json reports/report2.json
```

### Delete Old Reports
```bash
# Keep only last 5 reports
python3 reports_manager.py clean 5

# Keep only last 10 reports
python3 reports_manager.py clean 10
```

### Export Summary to CSV
```bash
python3 reports_manager.py export
```

## Best Practices

1. **Regular Analysis**: Run analysis before commits
2. **Track Progress**: Keep reports to see improvements
3. **Compare Results**: Use compare feature to track changes
4. **Clean Up**: Delete old reports periodically

## Automation

### Run Analysis Daily
```bash
# Add to crontab
0 2 * * * cd /path/to/project && pulse comprehensive .
```

### Keep Last 10 Reports
```bash
# Add to weekly cleanup
0 0 * * 0 cd /path/to/project && python3 reports_manager.py clean 10
```
